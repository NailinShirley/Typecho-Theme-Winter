## 如何自定义分类目录背景
在该分类下的分类描述填入如下内容:
> 分类描述$背景图片地址
> 一定要用$符号隔开!
> 
## 友链页面
规则如下:
```
#格式一定要对，否则出错
- 博客名|[博客地址](博客地址)|头像地址|博客简介
```
